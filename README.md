# Windows rcmd (AutoHotkey v2 App Switcher)

A lightning-fast, highly customizable application switcher for Windows, built with AutoHotkey v2. 

Inspired by the seamless `rcmd` experience on macOS, this script allows you to instantly switch between or launch your most-used apps and Chrome PWAs using the `Right Alt` key. If the app is already open, it brings the window to the front. If it's closed, it launches it instantly.

## ✨ Features
* **Rapid App Switching**: Press a single hotkey to toggle your workflow context.
* **Chrome PWA Support**: Reliably targets Chrome-installed web apps (Gemini, YouTube Music, NotebookLM) without getting confused by standard browser tabs.

🚀 Quick Start
Install AutoHotkey v2. (This script is not compatible with AHK v1).

Clone this repository or download the .ahk file.

Configure your paths.

Double-click the .ahk script to run it. A green "H" icon will appear in your system tray.

(Tip: To make this run automatically on boot, press Win + R, type shell:startup, and place a shortcut of the script in that folder.)
